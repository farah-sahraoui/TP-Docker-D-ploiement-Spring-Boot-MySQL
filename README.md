# TP Docker - Spring Boot & MySQL

## description:

Ce projet a permis de conteneuriser une application Spring Boot et une base de données MySQL pour simuler un environnement de production.
En utilisant Docker Compose, nous avons orchestré le déploiement des deux services, configuré leur interconnexion via des variables d'environnement 
et validé le bon fonctionnement de l'API ainsi que la persistance des données dans le volume Docker.
